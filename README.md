# r2jp
## Japanese Community of radare2

r2jpは日本国内のradare2コミュニティです。
ここでは日本語限定でradare2に対してサポートのやり取り、例えば：
- 質問/ Q&A （質問があれば新規Issuesのチケットで対応）
- 開発（プラグイン、追加機能、アイデア）
- 日本語ドキュメンテーション
などをやります。

radare2ベテランのユーザーが便利な使い方、質問の回答、howto手順などをWikiで書きます。
